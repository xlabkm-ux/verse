# Knowledge Base Layout

The `knowledge/` folder is the local source of truth for project canon before files are uploaded into OpenAI Agent Builder.

## Recommended Subfolders

- `world_bible/` - rules of the world, metaphysics, factions, geography
- `characters/` - biographies, motivations, appearance, relationships
- `timeline/` - chronology, key dates, event order
- `chapters/` - previous chapter drafts or approved chapter summaries
- `notes/` - loose project notes, themes, fragments, research

## Naming Rules

- Use short, stable names in Latin characters where possible.
- Start file names with a category or subject.
- Prefer one topic per file.
- Keep canonical facts in dedicated files instead of burying them in mixed notes.

Examples:

- `character_anna_petrovna.md`
- `timeline_book1.md`
- `world_magic_rules.md`
- `chapter_03_summary.md`

## Upload Guidance

Before running Preview in Agent Builder:

1. Review the local files for outdated contradictions.
2. Upload the most relevant files for the current test.
3. Re-upload files after major canon edits so Builder has the latest version.

## Search Guidance

File Search works best when materials contain repeated canonical anchors:

- full character names
- location names
- formal titles
- important dates or sequence markers
- named events
